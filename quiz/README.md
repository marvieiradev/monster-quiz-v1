# Monster Quiz

## Sobre o projeto

Jogo em formato de quiz, Baseado no jogo "Monster Hunter" da Capcom. Onde o objetivo é acertar o nome correto dos monstros da franquia, de acordo com a sua imagem.

## Tecnologias Utilizadas

• React
• Javascript
• Tailwind CSS
